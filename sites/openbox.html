Die Installation erfolgt nicht komplett auf Kommandozeile,
sondern leicht grafisch.

Da in der Anleitung zum Installieren von Arch Linux die komplette
Installation beschrieben ist, erspare ich mir das hier.

Die leicht graphische Installation ist auch soweit selbst erklärend.
Hier wird das Passwort für den Benutzer "ROOT" gesetzt.
Danach legt man den eigentliche Benutzer an, und vergibt,

neben wenn erwünscht weiteren Optionen,

auch das Kennwort.
Nach einem Reboot ohne Installations DVD wird von der Festplatte gestartet.

So sieht das fertig Installierte Betriebssystem aus.

Ohne graphische Oberfläche.
Nun wird hier der XFCE Desktop installiert.
